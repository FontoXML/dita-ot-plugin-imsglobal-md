# org.imsglobal.md

Defines a DITA-OT plugin containing the IMS Global metadata schema.

## Installation

1. Checkout this repository in ```{DITA_OT_PATH}/plugins/org.imsglobal.md```
2. From ```{DITA_OT_PATH}``` run ```ant -f integrator.xml```
